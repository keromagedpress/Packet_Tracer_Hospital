**Academic Context:** This project was developed as part of the "Computer Networks" at Innovation University.
* **Faculty Support:** Special thanks to Professor "Ahmed Mohamed Nagm" for his guidance and insights throughout the semester.
Designed a multi-department hospital network in Cisco Packet Tracer featuring VLANs (Staff, Guest, Server Farm), inter-VLAN routing, Site-to-Site VPN, NAT, ACL Firewall, DHCP/DNS, VoIP, and an IAM Controller — simulating a real-world enterprise hospital environment.
